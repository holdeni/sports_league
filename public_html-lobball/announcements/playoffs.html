Playoffs - Updated: 22 Aug 2013

Official Schedule Posted

The details for the playoffs are now available for viewing under Schedules. You can view the consolidated schedule for all divisions or just a specific division. The schedules also show which game is next for the teams involved depending upon who wins or loses. When game results are submitted, the schedule will be automatically updated to help teams confirm when their next game will be.

The Game Identifier column indicates the division (A, B or C) and the game number within the division's playoffs (e.g 1, 2, 3, etc). As noted on the schedules, the home team for each game (except the Finals) is the team that finished better in the standings. The home team in the Finals is determined differently. In the first final game, home team is the team who has not yet lost. For the 2nd game, if required, the winner of the 1st game is now the home team.

Import scheduling note:

There are a couple of spots in the schedule were a team will be required to play a 2nd game in the same week. These were minimized as much as possible but each division required some games to be scheduled as such.

A Division

  • Winner of game A12 plays later in the same week (in game A13)

B Division

  • Winner of game B01 plays later in the same week (in game B05)
  • Winner of game B06 plays later in the same week (in game B08)
  • LOSER of game B13 plays later in the same week (in game B15)
  • Winner of game B14 plays later in the same week (in game B15)

C Division

  • Winner of game B12 plays later in the same week (in game B13)

Four innings shall constitute a complete game (when the game is shortened due to rain, lighting, etc). Any Finals game will be a full 8 innings, regardless of timing, since the start time for them will be 7pm.

Please make every effort to play your games in the timeslot scheduled. Rain outs will be rescheduled only if absolutely necessary (e.g. the presence of lighting, or heavy rain). Games MAY be rescheduled quickly meaning that a game not played one night may be rescheduled to be played the next night so please be aware. Additionally, games rescheduling may cause changes to other unplayed games so please check the schedules online regularly. We will try to keep league members informed as to schedule changes the best we can.

Regular season time restrictions apply for all playoff games unless the teams are tied at the end of a complete inning. Except to meet the minimum four inning game criteria no complete inning shall commence unless there is more than 10 minutes available prior to the starting time of the next scheduled game (or 11:00PM). In the event of a tie, and less than 10 minutes remaining, then the last team to be leading at the end of a complete inning shall be deemed the winner.

Game results will be reported online through the web like regular season games. Just use the report playoff game link under Team Actions.
